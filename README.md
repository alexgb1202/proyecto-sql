# 📊 Proyecto SQL – Base de Datos Shakila

## 📌 Descripción

Este proyecto consiste en el análisis de la base de datos **Shakila**, utilizando SQL para resolver un total de 64 consultas.

El objetivo principal ha sido demostrar el manejo de consultas SQL desde un enfoque práctico, aplicando distintos niveles de complejidad y respetando buenas prácticas de estructuración y claridad en el código.

---

## 🎯 Requisitos del proyecto cumplidos

A lo largo del desarrollo del proyecto se han trabajado los siguientes puntos:

- ✔ Manejo de la herramienta **DBeaver**
- ✔ Archivo del esquema de la BBDD proporcionada
- ✔ Consultas con una sola tabla
- ✔ Manejo de relaciones entre tablas (JOIN)
- ✔ Uso de subconsultas
- ✔ Creación y uso de vistas
- ✔ Uso de estructuras de datos temporales
- ✔ Aplicación de buenas prácticas
- ✔ Interpretación y entendimiento de los resultados obtenidos
- ✔ Correcta entrega y estructuración en GitHub

---

## 🛠 Herramientas utilizadas
